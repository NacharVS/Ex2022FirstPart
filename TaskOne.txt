Реализовать приложение на платформе WPF моделирующее торговую площадку.

1. Регистрация:	
	Поля:
	а) Имя
	б) Фамилия
	в) Дата рождения
	г) Логин
	д) Пароль
	е) Повтор пароля

2. Авторизация осуществляется путем ввода логина/пароля

3. Основные функции приложения это просмотр каталога товаров, добавление в корзину и покупка.
3.1 У каждого зарегестрированного пользователя есть открытый счет, все покупки осуществляются за счет средств на счете.
3.2 Порядок покупки: Выбор товаров > Добавление в корзину > Просмотр карзины > Покупка 

4. Данные пользователей хранятся в базе данных.
5. Данные о покупках храняться в базе данных в виде ИмяПокупателя, КупленныеТовары, ОбщаяСтоимость, дата покупки. 
6. Реализовать возможность просмотра статистики по покупкам.


Проект должен быть ззалит на гит, в репозиторий Ex2022FirstPart, в ветку с название Фамилия_Имя_Группа (написать своё!). 
Каждый пункт, по завершению, должен быть залит отдельным комитом.
